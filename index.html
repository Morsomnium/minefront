Minecraft
custom modpack installation

1. ⠀ Download minecraft Tlauncher
2. ⠀ Run it on version 1.12.2
3. ⠀ When you get minecraft main menu you can exit.
4. ⠀ Download actual mod-pack. (password: 1231)
5. ⠀ Also download latest forge version.
6. ⠀ (On this step you are searching minecraft source folder) In windows 'run' field write '%appdata%', then you need to find folder '.minecraft'.
7. ⠀ Import mods from downloaded archive in 4-th step to the .minecraft mods folder. Do the same with config folder.
8. ⠀ Open forge installer (5-th step), install and extract it to '.minecraft'.
9. ⠀ Run T-launcher again.

* ⠀ Here you can change default settings of launcher. If you have 64-bit OS it's recommended to install 64-bit java and select more RAM-memory to run minecraft.
For example, if you have 8GB RAM, it's better to set approximately 4096MB RAM.

10. ⠀ Select in forge versions the latest one (1.12.2-forge 1.12.2-14.23.4.2760).
11. ⠀ Run minecraft.
12. ⠀ In multiplayer add new server and enter address: looga.net:25003
13. ⠀ Enter to our server!
